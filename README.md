# tikauto-legal

Use these values in the TikTok app form:

1. Terms of Service URL: `https://<your-username>.github.io/tikauto-legal/terms.html`
2. Privacy Policy URL: `https://<your-username>.github.io/tikauto-legal/privacy.html`
3. Platforms: check **Desktop** (and **Web** if using OAuth redirect/callback URL)

This repository includes:

- `/terms.html`
- `/privacy.html`
